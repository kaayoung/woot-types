### 실행 결과 : 7 12

### 실행 이유

makeAdder 은 함수를 리턴하므로 add5와 add10은 함수인 것이다. 즉, add5는 y+5 함수인 것이고, add10은 y+10 함수다.
그리고 마지막에 add5 와 add10에 각각 변수를 넣었으므로 각각의 합이 출력된다.
